## [whatcable](https://github.com/darrylmorley/whatcable) #1

> 一款面向 macOS 和 iOS 设备的电缆连接管理工具，可自动识别网络接口并生成可视化拓扑。开发者将其定位为系统网络偏好设置的补充，解决日常布线排查与端口映射的痛点。🏆

<p><img src="https://opengraph.githubassets.com/1/darrylmorley/whatcable" alt="whatcable" style="max-width:100%;border-radius:8px;"></p>

随着 Apple Silicon Mac 和 iPad 在多显示器、外置存储及有线网络场景下的普及，用户经常需要在多个接口间切换。**whatcable** 利用原生 `IOKit` 框架实时监控 Thunderbolt、USB-C 及以太网口的连接状态，以列表和图形两种方式展示当前链路。

项目采用纯 **Swift** 编写，支持 macOS 14+ 和 iOS 17+，并通过 **SwiftUI** 构建统一界面。核心功能包括端口命名、连接时间记录以及日志导出，方便 IT 管理员快速定位网线松动或交换机故障。

上线两周即获得 **1122** 个 Stars、**24** 个 Forks，在 Hacker News 和 Apple 开发者社区受到关注。不少用户反馈其 CLI 模式可嵌入自动化脚本，批量检测多台机器的物理连接状态。

与系统自带 “网络” 面板相比，whatcable 更侧重 **物理层** 的诊断。它不修改网络配置，只做读取和可视化，因此在安全合规场景（如企业机房巡检）中可作为轻量级辅助工具使用。

相关链接：
- 项目地址：https://github.com/darrylmorley/whatcable

## [dbx](https://github.com/t8y2/dbx) #2

> 一个基于 **Vue 3** 构建的数据库管理前端面板，支持 MySQL、PostgreSQL 和 SQLite 三种引擎的实时查询与表结构编辑。项目旨在提供比 phpMyAdmin 更现代的交互体验。

<p><img src="https://opengraph.githubassets.com/1/t8y2/dbx" alt="dbx" style="max-width:100%;border-radius:8px;"></p>

传统数据库管理工具有两种路径：笨重的桌面客户端和功能有限的 Web 端。**dbx** 选择纯前端轻量化路线，后端仅需一个 Node.js 代理层，通过 REST API 与数据库通信。前端使用 **Vue 3** + **Pinia** 状态管理，支持暗色模式与可拖动布局。

核心功能包含 SQL 编辑器的语法高亮、自动补全、查询历史，以及可视化表结构设计器。项目当前支持 **688** 个 Stars、**35** 个 Forks，在 GitHub 上的更新频率较高，最近一周内修复了连接池泄漏问题。

与同类产品相比，dbx 的优势在于 **单页应用** 的零刷新操作和 **多数据源** 同时连接。用户可以在一个标签页内切换不同数据库，并通过书签保存常用查询，适合 DevOps 和全栈开发者日常调试使用。

相关链接：
- 项目地址：https://github.com/t8y2/dbx

## [SoftwareCopyright-Skill](https://github.com/Fokkyp/SoftwareCopyright-Skill) #3

> 一个 **Python** 库，用于检测源代码文件中可能存在的版权声明、许可证头部与第三方代码片段。它可作为 CI/CD 流程中的合规检查步骤，降低企业开源代码使用的法律风险。

<p><img src="https://opengraph.githubassets.com/1/Fokkyp/SoftwareCopyright-Skill" alt="SoftwareCopyright-Skill" style="max-width:100%;border-radius:8px;"></p>

随着开源生态的繁荣，软件供应链中的版权合规问题日益突出。**SoftwareCopyright-Skill** 由一位前知识产权律师与工程师联合开发，核心思路是将常见开源许可证的条款模式化为正则规则，并与 `ast`（抽象语法树）分析结合。

工具支持 C / C++、Java、Python、JavaScript 等主流语言的注释解析，可识别 **GPL、MIT、Apache 2.0** 等 20 余种许可证头部，并标记无版权声明的文件。项目当前收获 **599** 个 Stars、**136** 个 Forks，Fork 率高说明社区有定制扩展的需求。

在实际应用中，可以将其集成到 GitHub Actions 或 Jenkins Pipeline 中，每次提交时自动扫描新增代码。输出结果包含风险等级与建议操作，方便非法律背景的开发者理解。与依赖扫描工具（如 FOSSA）不同，它更专注于源码内嵌的版权文本而非依赖树。

相关链接：
- 项目地址：https://github.com/Fokkyp/SoftwareCopyright-Skill
