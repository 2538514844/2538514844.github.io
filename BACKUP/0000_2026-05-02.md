# [2025-04-12] GitHub 开源项目精选

## 概览
本期精选聚焦三个不同技术方向的小而美项目：一款轻量离线可方向键操作的2FA验证器、一个纯C语言实现的极简GPT训练推理框架、以及一个基于Kotlin+Ktor的全栈式后端项目模板。它们均处于早期阶段，但各自在易用性、原理教学或开发范式上展现出独特价值。

## 安全工具
- **[flipphoneguy/totp_app](https://github.com/flipphoneguy/totp_app)** ⭐6 — 完全离线、支持方向键操作的轻量级TOTP（基于时间的一次性密码）Android应用，专为无触屏或低交互场景设计，隐私友好且节省电量。

## AI/大模型
- **[jmerelnyc/gpt-nucleus](https://github.com/jmerelnyc/gpt-nucleus)** ⭐6 — 从零开始实现的最小化GPT训练与推理框架，使用纯C语言编写，适合希望深入理解Transformer底层原理的学习者。

## 开发框架
- **[Dev-moe-kyawaung/supreme-tachyon-ktor](https://github.com/Dev-moe-kyawaung/supreme-tachyon-ktor)** ⭐6 — 基于Kotlin + Ktor构建的全栈式后端项目模板，集成了常用中间件与认证逻辑，为快速搭建RESTful API提供开箱即用的基础设施。
